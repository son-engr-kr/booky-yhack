# Booky 데모 녹화 스크립트

## 녹화 전 준비
- 백엔드 실행: `cd backend && .venv/Scripts/python -m uvicorn app.main:app --reload --port 8001`
- 프론트엔드 실행: `cd frontend && npm run dev`
- MongoDB 실행 + 시드 데이터 확인
- 브라우저 `http://localhost:3000` (모바일 뷰포트 ~390px 권장)

---

## 1. 홈 — 행성 뷰 (`/planet`)
- [ ] 궤도 태양계 보여주기 — 내 행성 중앙, 친구들 공전
- [ ] 친구 이름 + 유사도 % 설명
- [ ] 말풍선 순환 기다리기 (친구들의 최근 독서 활동)
- [ ] **친구 행성** 탭 → 친구 상세로 이동

## 2. 친구 행성 상세 (`/planet/friend/{userId}`)
- [ ] 친구 행성 이미지 보여주기 (AI 생성)
- [ ] 통계 강조: 읽은 책, 노트, 매치율 %
- [ ] "함께 읽는 책" 스크롤
- [ ] "별자리 보기" 탭 → 별자리 페이지
- [ ] 홈으로 돌아가기

## 3. 행성 상세 / 프로필 (`/planet/detail`)
- [ ] 홈에서 **내 행성** 탭
- [ ] 2x2 통계: 책, 노트, 선택, 진행률
- [ ] **장르 도넛 차트** 스크롤
- [ ] **읽는 중** 프로그레스 바 + 로켓 이모지
- [ ] **"My Reading Notes"** 탭으로 전환
- [ ] 생성된 독서 노트 보여주기 (탭해서 펼치기)
- [ ] 3파트 리포트: 나의 독서, 친구들의 독서, 스토리 브랜치
- [ ] 돌아가기

## 4. 라이브러리 (`/library`)
- [ ] 책 그리드 — 표지, 제목, 저자
- [ ] 안 읽은 책에 "NEW" 뱃지 설명
- [ ] 읽는 중인 책에 프로그레스 바 설명
- [ ] **위대한 개츠비** 탭

## 5. 책 상세 (`/library/great-gatsby`)
- [ ] 표지, 제목, 저자, 장르 뱃지
- [ ] 진행 바 (챕터 X / 9, Y%)
- [ ] "함께 읽는 친구" 아바타
- [ ] **"Story Recap"** 탭 → 6컷 만화 리캡 (AI 생성 패널)
- [ ] 리캡 닫기
- [ ] **"Reading Notes"** 탭 → AI 독서 리포트
  - [ ] 나의 독서 요약
  - [ ] 친구들의 독서 요약
  - [ ] 스토리 브랜치 시각화 + 투표 통계
- [ ] 노트 닫기
- [ ] 3개 액션 버튼: 노트, 등장인물, Booky 채팅
- [ ] **"Continue Reading"** 탭

## 6. 독서 페이지 (`/library/great-gatsby/read?chapter=X`)
- [ ] 페이지 넘기기 방식 텍스트 (스크롤 없음)
- [ ] 페이지 상단 **캐릭터 칩** 설명 (하나 탭)

### 6a. 등장인물 팝업
- [ ] **초상화 삽화** 보여주기 (Imagen AI 생성)
- [ ] 이름, 역할, 스포일러 없는 설명
- [ ] "Ch.1–X 기반 (스포일러 없음)" 뱃지
- [ ] 팝업 닫기

### 6b. 친구 하이라이트
- [ ] **"Friends"** 토글 ON
- [ ] 텍스트 위 색깔 하이라이트 오버레이
- [ ] 친구 하이라이트 hover/탭 → 이름 + 코멘트 툴팁
- [ ] 하이라이트 탭 → 상세 팝업 + 댓글

### 6c. 하이라이트 생성
- [ ] **텍스트 선택** → 툴바 표시: "✦ Highlight" + "🎤 Ask"
- [ ] **Highlight** 탭 → 코멘트 추가 → 저장
- [ ] 텍스트에 amber 밑줄로 하이라이트 표시 확인

### 6d. 음성 질문 (Gemini)
- [ ] 텍스트 선택 → **🎤 Ask** 탭
- [ ] 구절에 대한 질문 말하기
- [ ] 듣기 애니메이션 → 질문 텍스트 변환
- [ ] "Gemini is thinking..." → 답변 표시
- [ ] 답변이 **TTS로 읽혀짐**
- [ ] "Ask another question" 또는 닫기

### 6e. 하이라이트 상세
- [ ] 내 하이라이트 탭 → 팝업
- [ ] 좋아요, 친구 댓글 보기
- [ ] 댓글 추가
- [ ] 내 댓글 삭제 (✕)
- [ ] 좋아요/취소
- [ ] 하이라이트 삭제

### 6f. AI MC 질문
- [ ] 다음 페이지로 이동 → **Booky 질문** 팝업
- [ ] 구절에 대한 생각을 자극하는 질문
- [ ] 답변 입력 또는 스킵

### 6g. 디트로이트 스타일 선택 (챕터 끝)
- [ ] 선택 포인트가 있는 챕터 마지막 페이지로 이동
- [ ] **"⚡ Choice Point"** 버튼 (깜빡임)
- [ ] 탭 → 디트로이트 스타일 모달
  - [ ] 질문 + 상황 설명
  - [ ] 옵션 선택
  - [ ] 투표 결과: 퍼센티지 바
  - [ ] 친구들이 뭘 골랐는지 표시
- [ ] "Continue to Chapter X →" 탭

### 6h. 페이지 네비게이션
- [ ] 좌우 스와이프로 페이지 넘기기
- [ ] Prev/Next 버튼 사용
- [ ] 챕터 시작: "← Ch.N"으로 이전 챕터
- [ ] 챕터 끝: "Next Chapter →" 또는 선택 포인트

## 7. 사이드 패널 (독서 페이지에서)

- [ ] 등장인물 아이콘 탭

- [ ] 대화 이어가기

## 강조할 AI 기능
1. **K2 Think V2**: 등장인물 설명, 독서 질문, 작가 채팅, 스토리 선택, 독서 노트, 리캡
2. **Gemini 2.0 Flash**: 구절에 대한 음성 Q&A (음성인식 → 답변 → 음성합성)
3. **Vertex AI Imagen**: 등장인물 초상화, 만화 패널, 행성 이미지

## 데모 팁
- **행성 뷰**로 시작 — 시각적으로 가장 인상적
- **독서 페이지**에 기능이 가장 많음 — 여기에 시간 많이 투자
- **음성 질문**을 와우 포인트로 보여주기
- **독서 노트**로 마무리 — 친구와 나의 독서 비교
- 예상 녹화 시간: **8-12분**
